# Ottawa Garage Door Website

A responsive React website for a local garage door repair company in Ottawa.

## Live Demo

[https://your-netlify-link.netlify.app/](https://ottawagaragedoor.netlify.app/)

## Features

- React + Vite
- React Router
- Responsive design
- Service pages
- Customer reviews carousel (Swiper)
- FAQ accordion
- Contact form
- Modern UI with React Icons

## Technologies

- React
- React Router
- Swiper
- React Icons
- CSS3
