# Ottawa Garage Door

A modern, responsive React website for a local garage door company serving Ottawa and surrounding areas.

## Live Demo

[https://your-netlify-link.netlify.app/](https://ottawagaragedoor.netlify.app/)

## Features

- Responsive design
- React Router navigation
- Dropdown navigation menu
- Service cards
- Customer reviews
- FAQ accordion using React Hooks (`useState`)
- Contact page
- Service Areas section
- Modern dark UI with orange accents

## Built With

- React
- JavaScript (ES6)
- React Router
- React Hash Link
- React Icons
- CSS3
- Vite
